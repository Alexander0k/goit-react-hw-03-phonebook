# goit-react-hw-03-phonebook

Візьми своє рішення завдання з попередньої домашньої роботи і додай зберігання
контактів телефонної книги в localStorage. Використовуй методи життєвого циклу.

1.Під час додавання та видалення контакту контакти зберігаються у
локальнесховище. 2.Під час завантаження застосунку контакти, якщо такі є,
зчитуються з локального сховища і записуються у стан.
